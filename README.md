# Red List Index analysis

These files are used for the Red List Index analysis.

Step 1. Prepare spatial datasets  
Download bird species range maps from BirdLife International, and download island and continental mainland shapefiles from the 30 m global land dataset.

Step 2. Calculate spatial overlap  
Overlay species range maps with island and mainland polygons. For each species, calculate island range area, mainland range area, total terrestrial range area, and the proportion of terrestrial range occurring on islands and mainland.

Step 3. Classify species distributions  
Classify species into strict island-endemic, strict mainland, island-dominant shared, balanced island–mainland shared, mainland-dominant shared, and ambiguous groups. The main classification is based only on terrestrial overlap; ocean overlap is recorded separately as an additional label.
